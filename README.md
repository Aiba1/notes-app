Mobile application for organizing study notes 

Aim 
To explore current applications that students use to organize their study notes and to use these insights to design a mobile note-taking application that better supports their learning needs.
`Objectives 
1.	Understand how students currently create, store, and retrieve study notes across courses.
2.	Identify pain points.
3.	Investigate what features would help students during study and exam prep.
4.	Translate findings into design requirements and MVP features for the planned mobile application.
